A common way to add plugin's to a Top-level navigation in WordPress. Introduced to make adding plugins to a universal location in WordPress 2.7.

== Description ==

This plugin is discontinued due to unforeseen issues with WordPress, Any plugin requiring MultiPlug has since been updated and no longer relies on this hunk of junk!

MultiPlug is a small plugin which allows other plugins, preferably ones made at Meandmymac.net by Arnan de Gans, to hook into the WordPress dashboard. Introduced to make adding plugins to a universal location in WordPress 2.7 more easy and logical.

== Installation ==

1. Unzip the file and upload the folder to your /wp-content/plugins/ folder.
1. Activate from the plugins page and See the new menu appear in the sidebar.
